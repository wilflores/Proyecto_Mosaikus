ISO 22000:2005 Seguridad Alimentaria


Orientada a definir y especificar los requerimientos para la implementación de un sistema de gestión para la inocuidad de los alimentos, tiene como objetivo primordial la seguridad alimentaria durante el transcurso de toda la cadena de suministro, mediante la incorporación de elementos de las Buenas Prácticas de Fabricación y el sistema de análisis de peligros y puntos críticos de control. Este sistema de gestión puede permitir a la organización demostrar que los productos que suministra cumplen con los requisitos establecidos por el cliente, como también los requerimientos reglamentarios que le aplican por seguridad alimentaria.
